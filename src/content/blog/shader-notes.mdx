# Shader Notes for the Hero

The visual language of the landing is heavily influenced by shader configuration.

## Useful parameters

- shape controls the base pattern.
- type changes dithering matrix style.
- speed and scale define movement and visual density.

Keeping these values intentionally constrained helps maintain a coherent brand feel.
